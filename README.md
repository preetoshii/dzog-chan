# Dzog-chan

A minimalist AI meditation app powered by OpenAI, embodying Dzogchen wisdom in 7 words or less.

## Setup

1. Add your OpenAI API key to `.env`:
   ```
   VITE_OPENAI_API_KEY=your_actual_api_key_here
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the app:
   ```bash
   npm run dev
   ```

## Features

- **Minimalist Design**: Clean black/white interface with dark mode toggle
- **Text Input**: Type your questions to the Dzogchen master
- **Voice Input**: Click the microphone icon to speak (converted to text)
- **AI Responses**: Powered by GPT-4, limited to 7 words max
- **Fade-in Animation**: Responses appear with a gentle fade effect

## Usage

Simply type or speak your question, and the Dzogchen master will respond with profound wisdom in 7 words or less, pointing to the natural state of awareness.
